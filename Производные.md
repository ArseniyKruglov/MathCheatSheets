| Функция        | Производная             |
|:--------------:|:-----------------------:|
| $c$            | $0$                     |
|                |                         |
| $x$            | $1$                     |
|                |                         |
| $x^{n}$        | $n \times x^{n - 1}$    |
| $\sqrt{x}$     | $\dfrac{1}{2 \sqrt{x}}$ |
| $\dfrac{1}{x}$ | $-\dfrac{1}{x^2}$       |
|                |                         |
| $\sin{x}$      | $\cos{x}$               |
| $\cos{x}$      | $-\sin{x}$              |
| $\tg{x}$       | $\dfrac{1}{\cos^2{x}}$  |
| $\ctg{x}$      | $-\dfrac{1}{\sin^2{x}}$ |
|                |                         |
| $e^x$          | $e^x$                   |
| $a^x$          | $a^x \times \ln{a}$     |
|                |                         |
| $\ln{x}$       | $\dfrac{1}{x}$          |
| $log_a x$      | $\dfrac{1}{x\ln{a}}$    |
